# Visão do Projeto

Como o projeto devia ser feito em grupo, minha equipe optou por desenvolver um aplicativo iOS.
Nossa visão para o app era criar uma interface atrativa para usuário não apenas desfrutar das funcionalidades, mas também se divertir.

![](/mobile-application.png)

Com isso, fica fácil entender como os aparelhos gastam energia, podendo ser substituidos por outros mais econômicos. Assim, a sua conta de luz chega mais barata e o meio ambiente agradece.
